## Getting started
First, make sure you have Elm 0.18 installed.
Then run the following commands:

```
git clone https://github.com/bryanjenningz/elm-undo-todos.git
cd elm-undo-todos
elm-make src/Main.elm --output dist/elm.js
open dist/index.html
```
